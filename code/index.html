These are the code libraries I have generated using APIMATIC using the Swagger definition for my dictionary API. I will update the SDKs as I add new endpoints to each API, but sometimes my adding of endpoints moves faster than the time I have for generating SDKs.

If you need any specific integration code, or help please use the issue management on the Github repository for the API. You can also take the Swagger definition for this API, and go to APIMATIC and generate your own libraries.
